# Description #

A collection of icons, configurations and scripts for dzen2.

# Dependencies #

  * scripts/temperature 
    - [pymetar](https://aur.archlinux.org/packages/pymetar/)
  * conkyrc
    - "$DZEN_DATA/icons" should point to the icons directory
